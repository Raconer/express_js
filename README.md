# express_js

> 현재 프로젝트는 ExpressJS(JavaScript)로 개발을 하며 ExpressJS의 가장 기본적인 설정을 한 프로젝트 입니다.

## Git 전략

* master
  * 가장 자주 사용될거 같은 기능 들만 모아 두며 기본적인 QA가 끝났을때 배포
* hotfix
  * master에 배포하기 전에 테스트 하는 Branch
* develop
  * QA 전 개발
* develop/feature
  * 기능이 겹치면 따로 정의 하는 Branch
  * ex) JPA, MyBatis
* feature
  * 기능 별 개발시 사용 Branch
